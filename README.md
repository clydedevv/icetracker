## How to Use

### View the Map

Go to **[ice.clydedev.xyz](https://ice.clydedev.xyz)** to see real-time ICE activity reports.

### Get Alerts

1. Open Telegram and message [@icetracker_msp_bot](https://t.me/icetracker_msp_bot)
2. Send `/alerts 55401` (replace with your zip code)
3. You'll get a DM whenever ICE is reported within 5 miles

**Customize your radius:**
```
/alerts 55401 10    ‚Äî alerts within 10 miles
/alerts off         ‚Äî stop alerts
/alerts status      ‚Äî check your subscription
```

### Join the Channel

Get all reports broadcast in real-time: [Join Alert Channel](https://t.me/+EnMv3G3j241jYjU0)

## Report a Sighting

### Via Telegram (Fastest)

1. Message [@icetracker_msp_bot](https://t.me/icetracker_msp_bot)
2. Use `/register` to request verified reporter status
3. Once verified, submit reports:

```
/submit ACTIVE, Lake Street & Chicago Ave, Two ICE vehicles spotted
```

### Via Web

Click **"Report"** on [ice.clydedev.xyz](https://ice.clydedev.xyz)
